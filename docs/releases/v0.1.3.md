# KoalaBye v0.1.3

Released 2026-06-13.

## UI and UX

- central English, German, and Spanish language registry with local flag assets
- responsive authenticated navigation with visible admin and source links
- clearer instance administration cards for users, organizations, campaigns, safety limits, and audit activity
- campaign public-page preview and copy controls for public URLs and browser-extension snippets
- localized required and optional form-field badges
- improved dark-theme contrast, responsive spacing, and status treatment

## Privacy and Diagnostics

- public campaign pages disclose enabled collection
- opt-in allowlisted URL context for app, extension, platform, source, channel, and UTM values
- raw query strings and unknown context keys are discarded
- authorized CSV and JSON exports can include sanitized linked context
- partial responses, text draft autosave, browser major versions, device classes, and UTC offsets remain uncollected

## Operations

- refreshed README quick start, project badges, privacy summary, and asset attribution
- release verification includes formatting, tests, vet, static policy checks, generated-code checks, and a production Docker build
